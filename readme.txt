Для викростання потрібно підключити файли diff.js та light.css.

Функція window.diff() визивається без параметрів.
У файлі light.css містяться стилі для відображення однакових/змінених/доданих/видалених елементів.

Підсвічуються такі елементи:
    - елементи однакові(теги, вміст), вони підсвічуються синім кольором;
    - теги елементів однакові, але вміс різний, вони підсвідчуються жовтим кольором та вважаються зміненими;
    - елемент який є у блокі AFTER, але відсутній у BEFORE, підсвічується зеленим кольором та вважається доданим;
    - елемент який був у блокі BEFORE, але відсутній у блокі AFTER, підсвічується червоним кольором та вважається видаленим.
